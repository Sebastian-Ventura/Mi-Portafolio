📘 JavaScript Básico

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Se ejecuta en el navegador y puede manipular elementos HTML, responder a eventos, validar formularios, y mucho más.

2. Conceptos Básicos

Concepto Descripción Ejemplo
Variables Almacenan datos para usarlos luego let nombre = "Ana";
Tipos de datos Texto, números, booleanos, arrays, objetos let edad = 20; let activo = true;
Funciones Bloques de código que ejecutan tareas function saludar() { alert("Hola"); }
Eventos Acciones que disparan código, como clicks button.onclick = function() { alert("Click"); };

3. Declarar Variables

En JavaScript, puedes declarar variables usando var, let o const. Aquí la diferencia:

Palabra clave Alcance (Scope) ¿Puede cambiar?
var Global o función Sí
let Bloque Sí
const Bloque No, es constante

5. Estructuras de Control

Permiten controlar el flujo del programa.

Estructura Descripción Ejemplo
Condicional if Ejecuta código si una condición es verdadera if(edad >= 18) { alert("Mayor de edad"); }
Bucle for Repite código varias veces for(let i=0; i<5; i++) { console.log(i); }

6. Tipos de Datos Comunes

  • String: Texto, ejemplo: "Hola Mundo"
  • Number: Números enteros o decimales, ejemplo: 42, 3.14
  • Boolean: Verdadero o falso, ejemplo: true, false
  • Array: Lista ordenada de elementos, ejemplo: [1, 2, 3]
  • Object: Colección de pares clave-valor, ejemplo: {nombre: "Ana", edad: 20}

7. Comentarios en JavaScript

Para agregar notas o explicar código sin afectar la ejecución:

8. Consola de JavaScript

Herramienta para imprimir mensajes y depurar código.

Ejemplo:

console.log("Hola consola");

Abre la consola en el navegador con F12 o Ctrl+Shift+I y ve la pestaña "Console".
