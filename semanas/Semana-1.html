📚 Semana 1: Codigo Emmet

1. ¿Qué es Emmet?

Emmet es una herramienta o plugin para editores de código (como Visual Studio Code, Sublime Text, Atom, entre otros) que permite escribir código HTML y CSS más rápido usando abreviaciones. Convierte esas abreviaciones en código completo automáticamente, ahorrando mucho tiempo y esfuerzo.

2. ¿Para qué sirve Emmet?

  • Generar estructuras HTML completas con pocas teclas.
  • Crear rápidamente selectores y reglas CSS.
  • Acelerar la escritura y reducir errores al codificar.
  • Mejorar la productividad en el desarrollo web.

3. ¿Cómo usar Emmet?

Generalmente, escribes una abreviatura en el editor y presionas la tecla Tab para que Emmet expanda la abreviatura en código completo.
Ejemplo: Escribe ul>li*3 y presiona Tab.

Pasos básicos para usar Emmet:

  1. Abre tu editor de código compatible con Emmet.
  2. Escribe una abreviatura Emmet válida.
  3. Presiona Tab o la tecla asignada para expandir.
  4. Observa cómo se genera automáticamente el código completo.

4. Tabla de abreviaciones comunes en Emmet

Abreviación Descripción Resultado Expandido
div Etiqueta div simple 
<div></div>
ul>li*3 Lista ul con 3 li 
<ul>
  <li></li>
  <li></li>
  <li></li>
</ul>
nav>ul>li.item$*4 Menú nav con lista y 4 elementos con clase 
<nav>
  <ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
  </ul>
</nav>
header>h1{Título} Etiqueta header con título dentro 
<header>
  <h1>Título</h1>
</header>

5. Ejemplos Prácticos de Emmet

Ejemplo 1: Crear un formulario simple con campos de texto y botón

form>label{Nombre}:input[type="text"]+label{Email}:input[type="email"]+button{Enviar}

Se expande a:

<form>
  <label>Nombre</label>
  <input type="text" />
  <label>Email</label>
  <input type="email" />
  <button>Enviar</button>
</form>

Ejemplo 2: Crear un artículo con título, párrafo e imagen

article>h2{Título}+p{Texto del párrafo}+img[src="img.jpg" alt="Imagen"]

Se expande a:

<article>
  <h2>Título</h2>
  <p>Texto del párrafo</p>
  <img src="img.jpg" alt="Imagen" />
</article>

6. Ventajas de Emmet

  • Aumenta la velocidad de escritura del código.
  • Reduce errores comunes al codificar.
  • Mejora la productividad y flujo de trabajo.
  • Compatible con muchos editores de texto.
