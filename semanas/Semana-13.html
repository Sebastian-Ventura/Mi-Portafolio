Laravel (PHP) como Backend

Aprende a usar el framework Laravel con ejemplos prácticos

¿Qué es Laravel?

Laravel es un framework de desarrollo web en PHP que permite construir aplicaciones robustas, seguras y mantenibles usando el patrón MVC (Modelo-Vista-Controlador).

Instalación rápida

composer create-project laravel/laravel nombre-proyecto

Luego, inicia el servidor local:

cd nombre-proyecto
php artisan serve

Ejemplo 1: Ruta simple

Define esta ruta en routes/web.php:

Route::get('/saludo', function () {
    return "¡Hola desde Laravel!";
});

Ejemplo 2: Ruta + Controlador

Crear un controlador:

php artisan make:controller SaludoController

Dentro del controlador app/Http/Controllers/SaludoController.php:

public function mostrar() {
    return "Saludos desde el controlador";
}

Y la ruta en web.php:

Route::get('/saludo', [SaludoController::class, 'mostrar']);

Ejemplo 3: Conexión a base de datos

Configura tu conexión en .env:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=nombre_bd
DB_USERNAME=root
DB_PASSWORD=

Y usa Eloquent para traer datos (ejemplo con tabla "usuarios"):

use App\Models\User;

Route::get('/usuarios', function () {
    return User::all();
});
