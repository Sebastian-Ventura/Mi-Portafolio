Aprendiendo Jakarta EE y Spring Boot

Introducción, componentes y ejemplos prácticos

¿Qué es Jakarta EE?

Es una plataforma Java para aplicaciones empresariales robustas. Incluye APIs como Servlet, REST, CDI, JPA, entre otros.

Ejercicio 1: Servlet básico


@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
  protected void doGet(HttpServletRequest req, HttpServletResponse res) throws IOException {
    res.setContentType("text/html");
    res.getWriter().println("<h1>Hola desde Jakarta EE</h1>");
  }
}

Ejercicio 2: REST API con Jakarta


@Path("/api")
public class SaludoService {
  @GET
  @Produces(MediaType.TEXT_PLAIN)
  public String hola() {
    return "Hola desde Jakarta REST";
  }
}

¿Qué es Spring Boot?

Es un framework para crear aplicaciones Java modernas rápidamente. Basado en Spring, con configuración mínima.

Ejercicio 1: Hello Controller


@RestController
public class HelloController {

  @GetMapping("/hello")
  public String hello() {
    return "Hola desde Spring Boot";
  }
}

Ejercicio 2: API con parámetros


@RestController
@RequestMapping("/saludo")
public class SaludoController {

  @GetMapping("/{nombre}")
  public String saludar(@PathVariable String nombre) {
    return "Hola " + nombre + ", bienvenido a Spring!";
  }
}
