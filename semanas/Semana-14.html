Introducción a Flask (Python)

Aprende a usar Flask como backend de forma sencilla

¿Qué es Flask?

Flask es un micro-framework escrito en Python, muy utilizado para crear APIs y aplicaciones web ligeras y potentes. Es minimalista, fácil de aprender y extensible.

Para instalarlo:

pip install flask

Ejemplo 1: Hola mundo en Flask

Este código crea una ruta que devuelve un mensaje de saludo:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "¡Hola, mundo desde Flask!"

if __name__ == "__main__":
    app.run(debug=True)

Para probarlo, guarda el archivo como app.py y ejecuta python app.py.

Ejemplo 2: API con JSON

Este ejemplo retorna un objeto JSON:

from flask import Flask, jsonify

app = Flask(__name__)

@app.route("/api/datos")
def datos():
    return jsonify({"nombre": "Juan", "edad": 30, "lenguaje": "Python"})

if __name__ == "__main__":
    app.run(debug=True)
