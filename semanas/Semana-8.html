📦 Ejercicios con React + Vite

Aprende a usar React con el entorno de desarrollo ultrarrápido Vite

🚀 ¿Qué es Vite?

Vite es una herramienta de construcción (build tool) que permite crear aplicaciones modernas de JavaScript como React, con recarga rápida (hot reload) y configuración mínima.

Para iniciar un proyecto:

npm create vite@latest mi-app -- --template react
cd mi-app
npm install
npm run dev

✅ Ejercicio 1: Contador

Este ejercicio muestra cómo usar useState para controlar un número que aumenta cada vez que haces clic.

📁 src/App.jsx


import { useState } from 'react'

function App() {
  const [contador, setContador] = useState(0)

  return (
    <div>
      <h1>Contador: {contador}</h1>
      <button onClick={() => setContador(contador + 1)}>Aumentar</button>
    </div>
  )
}

export default App

✅ Ejercicio 2: Lista de tareas

Aprende a trabajar con formularios y arrays en React.

📁 src/App.jsx


import { useState } from 'react'

function App() {
  const [tarea, setTarea] = useState('')
  const [tareas, setTareas] = useState([])

  const agregarTarea = () => {
    if (tarea.trim() === '') return
    setTareas([...tareas, tarea])
    setTarea('')
  }

  return (
    <div>
      <h1>Lista de Tareas</h1>
      <input
        value={tarea}
        onChange={(e) => setTarea(e.target.value)}
        placeholder="Nueva tarea"
      />
      <button onClick={agregarTarea}>Agregar</button>

      <ul>
        {tareas.map((t, i) => (
          <li key={i}>{t}</li>
        ))}
      </ul>
    </div>
  )
}

export default App
