Introducción a PHP

Aprende los fundamentos de PHP con ejemplos prácticos

¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor ampliamente usado para desarrollar sitios web dinámicos.

Es compatible con múltiples bases de datos y se ejecuta dentro de un servidor web como Apache o Nginx.

Ejemplo 1: Mostrar texto con PHP

Este script muestra un mensaje en pantalla usando echo.

<?php
echo "¡Hola, mundo desde PHP!";
?>

Ejemplo 2: Variables y operaciones

En este ejemplo, usamos variables y hacemos una suma básica.

<?php
$numero1 = 10;
$numero2 = 20;
$suma = $numero1 + $numero2;

echo "La suma de $numero1 y $numero2 es: $suma";
?>
