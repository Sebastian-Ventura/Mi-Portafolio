📘React

1. ¿Qué es React?

React es una librería de JavaScript desarrollada por Facebook para construir interfaces de usuario (UI) de forma eficiente y declarativa. Permite crear componentes reutilizables que gestionan su propio estado, facilitando la creación de aplicaciones web dinámicas y rápidas.

2. Conceptos Clave de React

Concepto Descripción Ejemplo/Notas
Componente Bloque reutilizable de UI que puede ser función o clase Ejemplo: function Hola() { return <h1>Hola</h1>; }
JSX Extensión de sintaxis para escribir HTML dentro de JavaScript Ejemplo: <div>Hola Mundo</div>
Props Parámetros que se pasan a los componentes Ejemplo: <Saludo nombre="Ana" />
Estado (State) Datos internos que un componente puede manejar y actualizar Se usa useState en componentes funcionales

3. ¿Cómo usar React?

  1. Instalar Node.js y npm (gestor de paquetes)
  2. Crear un proyecto con create-react-app (herramienta oficial)
  3. Escribir componentes usando JSX
  4. Usar npm start para correr la aplicación localmente

Comando para crear proyecto React:

npx create-react-app mi-app

Luego:

cd mi-app
npm start

4. Ejemplo básico de un componente React

import React, { useState } from 'react';

function Contador() {
  const [contador, setContador] = useState(0);

  return (
    <div>
      <h1>Contador: {contador}</h1>
      <button onClick={() => setContador(contador + 1)}>Incrementar</button>
    </div>
  );
}

export default Contador;

Este componente muestra un contador que aumenta al hacer click en el botón.

5. Diagrama básico de flujo en React

Elemento Descripción Función
JSX Escribe la UI de forma declarativa Define la estructura visual
Componentes Construyen la UI en partes reutilizables Manejan props y estado
Estado (State) Datos que pueden cambiar con interacción Actualiza la UI cuando cambia
Eventos Acciones como clicks o inputs Disparan cambios en estado o comportamiento

6. Ventajas de usar React

  • Componentes reutilizables y fáciles de mantener
  • Actualización eficiente del DOM con Virtual DOM
  • Gran comunidad y ecosistema
  • Se puede usar con otras librerías y frameworks
