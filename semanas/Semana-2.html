📘 HTML Básico y Avanzado

1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Define la estructura del contenido web usando etiquetas como <h1>, <p>, <a>, <div>, etc.

2. ¿Para qué se usa HTML?

  • Diseñar la estructura básica de una página web
  • Insertar textos, imágenes, enlaces, videos
  • Definir encabezados, secciones, artículos
  • Base para estilos (CSS) y programación (JavaScript)

3. ¿Cómo se usa HTML?

Se escribe dentro de archivos .html y se ejecuta en un navegador.

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un párrafo</p>
  </body>
</html>

4. HTML Básico: Etiquetas Comunes

Etiqueta Descripción Ejemplo
<h1> Encabezado principal 
<h1>Título</h1>
<p> Párrafo de texto 
<p>Texto aquí</p>
<a href=""> Hipervínculo 
<a href="https://google.com">Ir a Google</a>
<img src=""> Imagen 
<img src="foto.jpg" alt="Foto">

5. HTML Avanzado: Semántica y Formularios

HTML5 introduce etiquetas semánticas que mejoran la organización del contenido.

  • <header>, <nav>, <main>, <article>, <section>, <footer>

Ejemplo completo:

<header>
  <h1>Mi Blog</h1>
</header>
<main>
  <article>
    <h2>Post 1</h2>
    <p>Contenido del post</p>
  </article>
</main>
<footer>© 2025</footer>

Formularios:

<form>
  <label>Nombre: </label>
  <input type="text" name="nombre">
  <button>Enviar</button>
</form>

6. ¿Con quién se complementa HTML?

Tecnología Descripción Ejemplo de uso
CSS Para aplicar estilos al HTML 
<p style="color:red">Texto rojo</p>
JavaScript Para agregar interactividad 
<button onclick="alert('Hola')">Click</button>
PHP / Node.js Para gestionar bases de datos y lógica de servidor PHP: echo "Hola";

7. Diferencias con otros lenguajes

HTML CSS JavaScript
Estructura del contenido Diseño y apariencia Interacción y lógica
Lenguaje de marcado Lenguaje de estilo Lenguaje de programación

8. Conclusión

HTML es el primer paso para aprender desarrollo web. A medida que se domina, se puede combinar con CSS y JavaScript para crear sitios modernos, dinámicos y atractivos.
