Sistemas Inteligentes con Python

Introducción y ejemplos básicos de Inteligencia Artificial usando Python

¿Qué es un sistema inteligente?

Un sistema inteligente es un software capaz de percibir, razonar, aprender y tomar decisiones. Estos sistemas se basan en ramas como la inteligencia artificial, el aprendizaje automático y la lógica computacional.

Ejemplo 1: Sistema experto básico

Este ejemplo toma decisiones simples basadas en reglas.

# sistema_experto.py
def diagnostico_fiebre(temp):
    if temp >= 38:
        return "Tienes fiebre, consulta al médico"
    else:
        return "No tienes fiebre"

print(diagnostico_fiebre(39))

Salida: Tienes fiebre, consulta al médico

Ejemplo 2: Clasificador de texto con machine learning (Scikit-learn)

Este ejemplo clasifica mensajes como positivos o negativos usando aprendizaje automático.

# clasificador.py
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

# Datos de entrenamiento
mensajes = ["me encanta este producto", "horrible experiencia", "excelente atención", "no me gustó", "lo odio"]
etiquetas = ["positivo", "negativo", "positivo", "negativo", "negativo"]

# Modelo
vectorizador = CountVectorizer()
X = vectorizador.fit_transform(mensajes)
modelo = MultinomialNB()
modelo.fit(X, etiquetas)

# Predicción
nuevo = vectorizador.transform(["me gustó mucho"])
print(modelo.predict(nuevo))

Salida esperada: ['positivo']
